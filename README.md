# Matcha Website

A full-stack e-commerce web application for a matcha-themed online store, built with a React frontend and a Node.js/Express backend.

This project allows users to browse products, view product details, and add items to a shopping cart through a clean, themed interface.

## Overview

Matcha Website is a practice full-stack project designed to explore modern web development concepts, including:
- frontend development with React
- routing with React Router
- API integration with a backend service
- cart state management
- custom UI styling with CSS
- serving local images and assets

The repository is structured with separate `frontend` and `backend` folders.

## Features

- Browse a list of products
- View individual product details
- Add items to cart
- Styled product cards and navigation bar
- Custom matcha-themed branding and design
- Frontend connected to backend API for product data

## Tech Stack

- React Router
